Quick Start

Installation

deno add @ethanke/lum-ui

Basic Usage

import { Button, Card } from "jsr:@ethanke/lum-ui@^0.1.0";

const html = Card({
  title: "Welcome",
  children: `${Button({ text: "Click me", variant: "primary" })}`
});
